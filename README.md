# Fake_News_Detection_ISOT
# About Project 
This project is a machine learning-based web application that classifies news articles as Real or Fake using Natural Language Processing (NLP) techniques.

# Features
• Text preprocessing and cleaning
• TF-IDF vectorization
• Machine learning model for classification
• Predicts whether news is real or fake

# Technologies Used
• Python
• Scikit-learn
• Pandas, NumPy
• Matplotlib, Seaborn

# Project Workflow

• Data Collection (ISOT Dataset)
• Data Cleaning & Preprocessing
• Feature Extraction using TF-IDF
• Model Training (Logistic Regression)
• Model Evaluation
• Prediction

# How to Run
Open the Jupyter Notebook (.ipynb file)
Run all cells
Enter your own news text
Get prediction (Real / Fake)

# Dataset 
ISOT Fake News Dataset
